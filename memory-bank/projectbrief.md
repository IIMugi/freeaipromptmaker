# Project Brief

Free AI Prompt Maker is a Next.js App Router site for composing text prompts for image-generation interfaces. The main experience is a local prompt builder; the image-to-prompt route is the only feature that sends a validated image to a configured provider.

The publisher surface has two layers:

- verified educational guides that may appear in the blog hub, sitemap, feeds, and internal recommendations;
- `needs-review` material that remains direct-link accessible but is excluded from indexing and discovery until source and claim review is complete.

The current product values are transparency, user control, static performance, accessible interaction, bounded data handling, and verifiable editorial claims. Generated drafts require human review. Advertising is disabled.

Current rebuild evidence lives in `docs/adsense-rebuild/`.
